
# PlataniumV3

PlataniumV3 aims to be a swiss knife for Fortnite modders
## Features

- Save account safely
- Dump AES keys
- Bypass UE5 signatures checks
- Detour URL
- Disable SSL
- Set proxy to attach any http debugger

## Installation

- Download the pre-built binary
- Start PlataniumV3Launcher.exe
- You're done
